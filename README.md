# Strange Loops
---
Strange Loops is a project I developed to explore my interests in dynamical systems. The name is drawn from Douglas Hofstadter's writing, and the idea that intelligence --- and potentially consciousness --- is the result of hierarchical and deeply nested self-referential feedback loops. The Lorenz attractor is visually and aesthetically emblematic of this idea, and this project allowed me to visually explore how deterministic parameter combinations can produce beautiful patterns in phase space. 

--- 
To run this project on your local machine, clone the repo and execute the following commands:

``` bash
cd StrangeLoops/backend && npm run dev
cd ../frontend && npm run dev
```
Then, open the development server on localhost. 

Have fun exploring!
